# Asset Review

This document records imported and generated assets used by the current Chai Empire prototype.

## Phase 2 Visual And Audio Identity

No external texture, sprite, model, or audio files were imported for the Phase 2 visual/audio pass.

Current assets are generated at runtime by `ChaiGamePresenter`:

| Asset | Type | Source | Disk file | Notes |
| --- | --- | --- | --- | --- |
| `Stall Art` hierarchy | Unity UI geometry | Procedural `Image` objects in `BuildStallArt()` | No | Kettle, stove, counter, customer silhouettes, and UPI QR prop. |
| `Chai Empire Procedural Circle` | `Texture2D`/`Sprite` | Generated by `GetCircleSprite()` | No | Used for round UI shapes, steam wisps, customers, and QR prop details. |
| `Steam Wisp A/B/C` | Unity UI geometry | Procedural `Image` objects animated by `AnimateSteam()` | No | Lightweight alpha, size, and position animation. |
| Location backdrop palettes | Color data | `GetLocationVisualPalette(locationId)` | No | Recolors stall backdrop, glow, and counter surfaces by highest unlocked location. |
| `Chai Empire Button Press` | `AudioClip` | Generated by `GetButtonPressClip()` | No | Short mono decaying sine click. |
| `Chai Empire Purchase` | `AudioClip` | Generated by `GetPurchaseClip()` | No | Mono generated success cue for upgrades. |
| `Chai Empire Unlock` | `AudioClip` | Generated by `GetUnlockClip()` | No | Mono generated success cue for locations. |
| Optional haptics | Device vibration | `Handheld.Vibrate()` | No | Controlled by in-session Settings toggle. |

## Licenses And Attribution

- Phase 2 visual and audio identity adds no third-party art or audio licenses.
- The decorative UPI QR prop is not a real payment code and does not include imported brand artwork.
- Existing third-party code under `Assets/ChaiEmpire/Runtime/ThirdParty/BreakInfinity` is outside this Phase 2 asset pass and remains governed by its bundled license file.

## Review Notes

- Runtime-generated assets avoid binary import churn while the prototype UI is still being shaped.
- Production art/audio can replace these procedural placeholders later without changing economy state or save data.
- If future work imports textures, sprites, fonts, audio, or models, record source URL or creator, license, modifications, Unity import path, and attribution requirements here before committing.
